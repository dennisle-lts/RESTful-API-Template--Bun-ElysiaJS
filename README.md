# RESTful API Template with Bun & ElysiaJS

## Overview
This template has nearly everything you will need to start developing a new RESTful API with Bun runtime. At the time of creating this template, ElysiaJS is the framework that has the best performance and is optimized for Bun runtime. One of the key features that made me choose this framework is that it has auto-generated OpenAPI definitions, and the performance is really good with Bun runtime.

This template provides a solid foundation for building production-ready APIs with modern tooling and best practices. It includes essential features such as request validation, error handling, and API documentation out of the box. Whether you're building a small microservice or a large-scale application, this template helps you get started quickly without sacrificing code quality or performance.

The combination of Bun's blazing-fast runtime and ElysiaJS's lightweight architecture ensures that your API can handle high throughput with minimal latency. The template is designed to be easily extensible, allowing you to add custom middleware, plugins, and routes as your project grows.

## Prerequisites
- [Bun](https://bun.sh/) v1.0 or higher
- Node.js knowledge (helpful but not required)
- Basic understanding of RESTful API concepts

## Getting Started

### Installation
Install dependencies:
```bash
bun install
```

### Running the Development Server
Start the development server with hot reload:
```bash
bun dev
```

### Running in Production
Build and run the production server:
```bash
bun start
```

## API Documentation
Once the server is running, you can access the auto-generated OpenAPI documentation at:
```
http://localhost:3000/openapi
```
